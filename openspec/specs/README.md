# Main Specs (Source of Truth)

This directory holds the canonical specs once at least one change has been merged via `sdd-archive`.

Layout:

```
specs/
└── {domain}/
    └── spec.md
```

Domains for this project will be discovered during the first `sdd-propose` (likely candidates: `site-shell`, `menu`, `home`).

Until the first archive, this directory may be empty.
